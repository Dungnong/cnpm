# Diagrams - PlantUML Code

Thư mục này chứa code PlantUML cho tất cả các sơ đồ trong bài tập module Thống kê.

## Cách import vào Visual Paradigm

1. Mở Visual Paradigm
2. Vào menu **Diagram** → **New Diagram**
3. Chọn loại sơ đồ tương ứng
4. Vào menu **Tools** → **Import PlantUML** (hoặc **File** → **Import** → **PlantUML**)
5. Dán nội dung file `.puml` vào và nhấn OK
